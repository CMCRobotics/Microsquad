# How to develop

* Install node and NPM
```
../../mvnw compile
```
* Update the path to include node
```
source source-path.sh
```
* Run the live server
```
npm run serve
```
* Run webpack in watch mode to recompile the Typescript code automatically
```
npm run webpack
```
